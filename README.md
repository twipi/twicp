## Running

You need [Nix][nix] with Flakes enabled. First, enter the Nix shell:

```sh
nix develop
```

Then, generate all the needed files and run the app using [Task][task]:

```sh
task
```

[svelte]: https://svelte.dev/
[nix]: https://nixos.org/
[task]: https://taskfile.dev/
